# СЭД на блокчейн системе
> СЭД для передачи документов между образовательным учереждением на технологии Блокчейн


# Установка и настройка проекта в VS Code

## 1. Установка Python

- Скачайте и установите Python 3.8 или выше с официального сайта: https://www.python.org/downloads/
- Убедитесь, что Python добавлен в PATH при установке

## 2. Настройка VS Code
   
- Установите VS Code: https://code.visualstudio.com/
- Установите расширение Python для VS Code через Marketplace
- Установите расширение Python Extension Pack (опционально)

## 3. Настройка виртуального окружения

Откройте терминал в VS Code (Terminal -> New Terminal) и выполните команды:

**Создание виртуального окружения**
```python -m venv venv```

**Активация виртуального окружения**
- Для Windows:
```venv\Scripts\activate```
- Для Linux/Mac:
```source venv/bin/activate```

## 4. Установка зависимостей
   
**Установка всех необходимых пакетов**
```pip install -r requirements.txt```

## 5. Запуск сервера
   
# Запустите сервер командой
```python server.py```

Сервер будет доступен по адресу: ```http://localhost:8080```

## 6. Структура проекта
------------------
- ```server.py``` - основной файл сервера
- ```requirements.txt``` - список зависимостей
- ```templates/``` - шаблоны HTML
- ```uploads/``` - папка для загруженных файлов


# Подключение к серверу в локальной сети

## 1. Настройка на компьютере с сервером
------------------------------------
### 1.1 Узнайте IP-адрес компьютера:
  - Windows: откройте командную строку (cmd) и выполните команду ipconfig
  - Linux/Mac: откройте терминал и выполните команду ifconfig

### 1.2 Настройка брандмауэра Windows:
  - Откройте ```"Брандмауэр Защитника Windows"```
  - Выберите ```"Дополнительные параметры"```
  - Выберите ```"Правила для входящих подключений"``` -> ```"Создать правило"```
  - Выберите ```"Для порта"```
  - Укажите TCP порт ```8080```
  - Разрешите подключение
  - Назовите правило ```"SecureDocFlow"```

## 2. Подключение с других компьютеров

### 2.1 Откройте браузер

### 2.2 Введите в адресной строке:
  ```http://IP-АДРЕС:8080```
  Пример: ```http://192.168.1.100:8080```

## 3. Проверка подключения

- Если страница загрузилась - подключение успешно
- Если страница не загружается:
  * Проверьте правильность IP-адреса
  * Убедитесь, что компьютеры находятся в одной сети
  * Проверьте настройки брандмауэра
  * Убедитесь, что сервер запущен
